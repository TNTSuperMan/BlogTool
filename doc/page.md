# ページファイル変換に関する仕様

これはパターンで分けられます。

1. ":p:text"

これはそのままのHTML要素を追加します。

この例では、pの部分が要素名、textが要素の中身です。

出力：
```
<p>text</p>
```
また、その次に"=class=text"というパターンがあれば、
なんていうんだろう。あれを追加します。
例：
1.page
```
:p:text
=class=text
```
出力
```
<p class="text"></p>
```
こんな感じで、まあ連続させることもできる。

2. ";t;text;text"

これは、config/temp.jsonに設定されたテンプレートを呼び出します。
